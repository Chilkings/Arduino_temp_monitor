# Arduino_temp_monitor
使用Arduino开发的温度检测系统

## 使用的器件

arduino uno *1

DS18b20 *4

蜂鸣器 *1

led灯 *1

0.96oled屏幕 *1

按键 *3

电阻若干



## 功能简介

使用了 4 个`DS18B20`对温度进行监测。可以手动调节报警阈值。当温度达到警报值时，蜂鸣器响，led亮。
